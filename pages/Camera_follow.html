using UnityEngine;

using UnityEngine;

public class CameraFollow : MonoBehaviour
{
    public Transform target;
    // Target object to follow（カメラが追いかける対象）

    public Vector3 offset = new Vector3(0, 5, -8);
    // Offset distance from the target（ターゲットとの距離）

    public float smoothSpeed = 10f;
    // Controls smooth movement speed（動きの滑らかさを調整）

    void LateUpdate()
    {
        // Runs after Update for stable camera motion（安定したカメラ動作のため）

        Vector3 desiredPosition = target.position + offset;
        // Calculates desired camera position（目標カメラ位置を計算）

        Vector3 smoothedPosition =
            Vector3.Lerp(transform.position, desiredPosition, smoothSpeed * Time.deltaTime);
        // Smoothly moves toward the target position（目標位置へ滑らかに移動）

        transform.position = smoothedPosition;
        // Applies the new camera position（計算結果を反映）
    }
}

This script makes the camera smoothly follow a target object, such as the player.
The camera keeps a fixed distance from the target using an offset and smoothly moves toward the desired position using linear interpolation.
LateUpdate() is used so the camera movement happens after the target has finished moving, resulting in more stable and natural camera behavior.

このスクリプトは、カメラがプレイヤーなどの対象オブジェクトを滑らかに追従する処理を行います。
ターゲットの位置に offset を加えることで一定の距離を保ち、
Vector3.Lerp を使って現在位置から目標位置へ少しずつ移動させています。
LateUpdate() を使うことで、ターゲットの移動後にカメラが更新され、より安定したカメラ挙動になります。
