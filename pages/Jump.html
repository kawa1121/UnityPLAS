using UnityEngine;

public class PlayerJump : MonoBehaviour
{
    public float jumpForce = 5f;
    private Rigidbody rb;
    // Component used for physics-based movement（物理挙動を扱うコンポーネント）

    void Start()
    {
        rb = GetComponent();
         // Gets a component from the same GameObject（同じ GameObject からコンポーネントを取得）
    }

    void Update()
    {
        if (Input.GetButtonDown("Jump"))
        {
             // Detects the moment a button is pressed（ボタンが押された瞬間を検出）

            rb.AddForce(Vector3.up * jumpForce, ForceMode.Impulse);
             // Applies force to a Rigidbody（Rigidbody に力を加える）
             // Applies force instantly like an impact（衝撃のように一瞬で力を加える）
        }
    }
}

This script allows the player to jump using physics.
When the jump button is pressed, an upward force is applied to the Rigidbody,
causing the GameObject to jump realistically with gravity.

(A Rigidbody is a Unity component that allows a GameObject to be affected by the physics engine.
It enables realistic movement such as gravity, falling, jumping, and collisions.
When a Rigidbody is attached, Unity automatically calculates forces, acceleration, and interactions with other objects.
Developers can move a Rigidbody by applying forces instead of directly changing its position.)

重力の影響を受けた自然なジャンプ動作が実現されます。
