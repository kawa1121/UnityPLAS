using UnityEngine;

public class SimpleGravityTransform : MonoBehaviour
{
    public float gravity = -9.81f;
    // Strength of gravity applied to the object（重力の強さ）

    float verticalSpeed = 0f;
    // Current vertical movement speed（現在の上下方向の速度）

    void Update()
    {
        verticalSpeed += gravity * Time.deltaTime;
        // Uses frame time to update movement smoothly（フレーム間の時間を使って更新）

        transform.position += Vector3.up * verticalSpeed * Time.deltaTime;
        //Upward direction vector in Unity（Unityでの上方向ベクトル）
        // Changes the object’s world position（ワールド座標を変更）

        if (transform.position.y <= 0f)
        {
            // Checks if the object has reached ground level（地面に到達したかを判定）

            Vector3 pos = transform.position;
            pos.y = 0f;
            transform.position = pos;

            verticalSpeed = 0f;
            // Stops vertical movement（落下を停止）
        }
    }
}

This script simulates simple gravity using Transform instead of physics.
The object continuously falls downward, accelerates due to gravity,
and stops when it reaches the ground level.
(A Transform is a Unity component that stores a GameObject’s position, rotation, and scale in the scene.
Every GameObject in Unity has a Transform component by default.
When you modify the Transform directly, the object moves immediately without using physics calculations.
This makes Transform useful for simple movements, UI elements, and objects that do not require realistic physics.)

オブジェクトは重力によって落下し、地面に到達すると停止します。

すべての GameObject は必ず Transform を持っています。
Transform を直接変更すると、物理演算を使わずに 即座にオブジェクトの位置が変化します。
そのため、簡単な移動や UI、物理挙動が不要なオブジェクトに向いています。）
